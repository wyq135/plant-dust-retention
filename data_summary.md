# 植物叶片滞尘数据集 —— 构建报告与初步分析

## 数据集概况

| 指标 | 数值 |
|------|------|
| 数据记录数 | 32 |
| 来源研究数 | 8 |
| 覆盖城市 | 7 (武汉/乌鲁木齐/兰州/北京/合肥/杭州/多城市综述) |
| 树种数 | 20 |
| 数据时间范围 | 2019-2023 |
| 所有单位 | 统一为 g/m² |

## 来源文献

1. **王琴等, 2020** — 武汉市15种阔叶乔木滞尘能力与叶表微形态特征. 生态学报. DOI: 10.5846/stxb201808241808
2. **李耀华等, 2022** — 城市园林树木叶面微结构特征对大气颗粒物滞留能力的影响. 生态学报. DOI: 10.5846/stxb202103080635
3. **李晓璐等, 2022** — 乔木滞留大气颗粒物能力及其与叶表面微结构关系. 中国城市林业
4. **甘肃农业大学学报, 2023** — 兰州市10种常用园林绿化树种叶表面微结构对其滞尘量的影响. DOI: 10.13432/j.cnki.jgsau.2023.04.021
5. **宋英石等, 2022** — 北京市9种常见绿化树种滞尘量及影响因子
6. **郑瑶瑶和李晓燕, 2019** — 不同城市11种常用绿化树种滞尘能力研究（跨城市综述）
7. **贺丹等, 2022** — 郑州市常绿树种滞尘能力与叶片生理结构的响应. DOI: 10.19303/j.issn.1008-0384.2022.002.010
8. **彭镇华(中国城市森林)** — 合肥15种植物滞尘数据

## 核心发现

### 1. TSP滞尘量总体分布

| 统计量 | TSP (g/m²) |
|--------|------------|
| 最小值 | 0.15 |
| 中位数 | 1.14 |
| 均值 | 1.70 |
| 最大值 | 5.27 |
| 变异系数 | 0.87 |

### 2. 不同土地利用类型的滞尘量差异

| 土地利用 | 均值 (g/m²) | 记录数 |
|----------|------------|--------|
| 工业区（兰州） | 1.96 | 4 |
| 城市混合（多城市综述） | 3.46 | 5 |
| 城市混合区 | 1.35 | 18 |
| 交通干道 | 1.02 | 5 |

**关键倾向**：多城市综述数据（基于大量权重平均）显著高于单城市实测，提示综述类数据可能存在选择偏差，应优先使用原始实测数据。

### 3. 跨城市同一树种的巨大差异

| 树种 | 城市间 CV | 最小 (g/m²) | 最大 (g/m²) |
|------|----------|-----------|-----------|
| 银杏 | 0.85 | 0.20（杭州） | 2.38（综述） |
| 女贞 | 0.92 | 0.83（武汉） | 3.92（合肥） |
| 二球悬铃木 | 0.55 | 0.55（杭州） | 1.89（武汉） |
| 香樟 | 1.11 | 0.43（武汉） | 3.60（综述） |

### 4. 环境因素相关性（重要发现）

| 环境因子 | 与TSP相关系数(r) |
|----------|:---------------:|
| 环境PM10浓度 | -0.20 |
| 环境PM2.5浓度 | -0.33 |
| 年均温度 | -0.03 |
| 年均湿度 | -0.22 |

**所有环境因子均呈弱负相关**——这违背直觉（按理高污染区植物应滞留更多灰尘）。可能原因：
- 不同研究的测量方法差异（采样周期、滤膜孔径）掩盖了环境信号
- 武汉/杭州等低污染城市的数据来自更规范的研究，测量值反而更高
- 兰州/乌鲁木齐等高污染城市的研究采样间隔可能更短

## 倾向与建议

### 适合构建大规模数据集的理由
1. ✅ 单位可统一（g/m²是最通用指标）
2. ✅ 植物类型可限定（乔木类数据充足）
3. ✅ 文献源充足（仅中文核心期刊就有30+篇相关研究）
4. ✅ 有跨城市对比价值

### 关键风险
1. ⚠️ **测量方法不统一**：采样间隔（雨后天数）是最大变异源，必须作为关键协变量记录
2. ⚠️ **环境数据稀疏**：大多数论文不提供完整的气象/污染背景数据
3. ⚠️ **正向推断困难**：需要严格控制的实验设计才能推断滞尘能力差异的原因

### 下一步建议
1. **优先获取**王琴等(2020)的武汉完整数据（含PM10/PM2.5分粒径），这是目前最完整的单研究数据集
2. **统一采样周期**：筛选"雨后7天"采样间隔的研究，或引入采样间隔作为校正因子
3. **扩展粒径数据**：有针对性地搜索含PM2.5/PM10/TSP三分级数据的论文
4. **加入叶面特征**：叶面绒毛密度、沟槽深度、蜡质含量等微观结构可解释滞尘差异
5. **用ASReview加速**：将100+篇检索结果导入ASReview，主动学习筛选高质量论文
